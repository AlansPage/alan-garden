# Rhizome

The rhizome is a botanical structure: think ginger, grass, iris. It grows horizontally. It has no taproot, no single point of origin. Any point can connect to any other point. It is acentered, non-hierarchical.

Deleuze and Guattari use it as a model for thought. The root-tree model dominates Western thinking: one truth, many branches. The rhizome model offers something else: many entries, no single trunk. A [[nomadic-consciousness|nomadic]] structure for thought.

In a rhizome, the middle is not an average—it is where things happen. Growth occurs through the middle. Connections proliferate. You cannot find the "original" idea because there is none. Just a mesh.

[[Apparatus-of-capture|The apparatus of capture]] prefers the tree. It wants a trunk to seize. A center to control. The rhizome frustrates capture because there is no center. Cut one connection, others remain. The structure is resilient through redundancy.

[[Desire-machines|Desire machines]] connect rhizomatically. Desire does not flow from a single source toward a single object. It spreads. It connects this to that, and that to something else. The [[body-without-organs]] is the smooth surface across which these connections slide.

I think of my [[anti-sisyphus|notes]] as rhizomatic. They link to each other without hierarchy. No note is the "main" one. Each is a node in a mesh. The [[deterritorialization]] of thought—the escape from the tree—happens when we write this way. When we let the links multiply.

The risk of the rhizome: chaos. No structure at all. Deleuze and Guattari are not proposing that. The rhizome has principles—connection, heterogeneity, multiplicity. It is organized differently, not disorganized. The difference matters.
